Demystifying Blockchain

Data visualization and website

Demystifying Blockchain is a project developed during the The Final Synthesis Design Studio course at Politecnico di Milano for the Master's Degree in Communication Design.
Demystifying Blockchain helps you understand the basics of blockchains and give a broader view of sustainability around this technology. The project is divided into two Phases:
→ Phase A is a report about the sustainable blockchain ecosystem
→ Phase B is a website that demystifies blockchain.
Moreover, the About page discloses the steps behind the project in details.


My skills coding, data visualization, infographics

Software and languages Figma, RawGraphs 2.0, Illustrator, InDesign, html, css, d3.js, p5.js
